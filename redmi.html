Redmi / Xiaomi

  1. Manage apps' battery usage

    Go to Settings → Battery → Manage apps' battery usage → Choose apps. Select AKonnect and set "No restrictions."














  2. Lock AKonnect in the "Recent apps"/ "App overview" pane.

    Open AKonnect. Now press the "Recent apps/Overview button" (that's the square button on stock Android, or the button with three horizontal lines). From there, select AKonnect, pull down on it to "lock it", so it never gets cleared.

  3. Enable Autostart

    Go to Settings Permissions → Autostart. Select AKonnect and toggle the switch to turn it on.









  4. Enable App Notifications

    Go to Settings → App Notifications → Select AKonnect and turn on the switch for "Priority".
